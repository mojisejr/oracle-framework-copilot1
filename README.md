# Oracle Framework

> "The Oracle Keeps the Human Human"

Complete Claude Code framework with commands, agents, and the psi (psi) folder structure.

## What's Included

| Component | Description |
|-----------|-------------|
| **CLAUDE.md** | Modular documentation hub |
| **Commands** | `/oracle`, `/rrr`, `/snapshot` |
| **Agents** | `oracle-keeper`, `context-finder` |
| **psi/ Structure** | 5-pillar AI brain organization |
| **Templates** | Retrospective, learning, confirmation |

## Quick Start

1. Clone this repo into your project
2. Restart Claude Code
3. Run `/oracle` to check alignment

## The psi/ (Psi) Structure

```
psi/
├── active/      <- Research in progress (gitignored)
├── inbox/       <- Communication, handoffs
├── memory/      <- Knowledge base
│   ├── retrospectives/
│   ├── learnings/
│   ├── logs/
│   └── resonance/
├── writing/     <- Blog drafts
└── lab/         <- Experiments
```

## Philosophy

Three pillars:
1. **Nothing is Deleted** - Append only, timestamps = truth
2. **Patterns Over Intentions** - Observe behavior, not promises
3. **External Brain, Not Command** - Mirror reality, don't decide
