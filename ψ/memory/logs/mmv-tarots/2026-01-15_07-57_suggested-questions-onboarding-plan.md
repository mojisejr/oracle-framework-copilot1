# Snapshot: Suggested Questions & Onboarding Ritual Plan

**Time**: 2026-01-15 07:57
**Context**: เตรียมแผนพัฒนาเฟสถัดไป (v1.1) สำหรับ mmv-tarots หลังจากสรุป v1 เสร็จสมบูรณ์

## Insight

ปัญหาที่พบคือผู้ใช้อาจเกิด **"Blank Page Anxiety"** เมื่อต้องเริ่มถามคำถามกับ AI ครั้งแรก ทำให้เกิดความประหม่าหรือไม่มั่นใจว่าควรจะถามอะไร หรือถามสั้นแค่ไหนถึงจะเหมาะสม

### กลยุทธ์การแก้ปัญหา (Dual-Approach):

1. **Suggested Question Chips (One-tap Wisdom)**:
   - เพิ่มปุ่มคำถามแนะนำเหนือช่อง Input (เช่น "วันนี้ไพ่อยากบอกอะไร", "การงาน 7 วันข้างหน้า")
   - เน้นความเร็ว (Low Friction) และการนำร่องคำถามที่มีประสิทธิภาพ (Prompt Quality Control)
   - แสดงผลเมื่อช่อง Input ว่างหรือเมื่อมีการ Focus

2. **Welcome Ritual Modal (Onboarding)**:
   - แสดง Modal ต้อนรับเมื่อ User เข้าใช้งานครั้งแรก (First Login)
   - ใช้เพื่อ "Set the Mood" แนะนำ Mimi ในฐานะผู้ชี้นำจิตวิญญาณ
   - อธิบายกติกาสั้นๆ (ระบบ Star และระบบพักหายใจ/Cooldown) เพื่อสร้าง Perception ว่านี่คือพิธีกรรมไม่ใช่แค่เครื่องมือแชท

## Apply When

- เมื่อเริ่มเปิดโปรเจค mmv-tarots เพื่อทำเฟสถัดไป (v1.1)
- เมื่อต้องการยกระดับ Conversion Rate ในขั้นตอนการถาม
- เมื่อพบความเห็นจากผู้ใช้ว่า "ไม่รู้จะเริ่มคุยกับ AI ยังไง"
