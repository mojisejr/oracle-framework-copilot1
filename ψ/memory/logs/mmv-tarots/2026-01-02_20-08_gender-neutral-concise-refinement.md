# Snapshot: Gender-Neutral & Concise Prompt Refinement

**Time**: 2026-01-02 20:08
**Context**: Refining the Mystic agent's prompt based on user feedback regarding gender neutrality and brevity.

## Insight

การออกแบบ AI Persona ที่ต้องปฏิสัมพันธ์กับผู้ใช้หลากหลายกลุ่ม จำเป็นต้องคำนึงถึง:
1. **Gender Neutrality**: หลีกเลี่ยงการระบุเพศในคำทักทาย (เช่น ห้ามใช้ "คนสวย", "คะ/ขา") เพื่อให้ครอบคลุมผู้ใช้ทุกเพศ (LGBTQ+, Men, Women) โดยเปลี่ยนไปใช้ภาษาที่เป็นกลางแต่ยังคงความสนิทสนม
2. **Brevity (Conciseness)**: การลดความยาวเนื้อหาลง 10% และตัดคำฟุ่มเฟือยช่วยให้คำทำนายดู "คม" และ "ฟันธง" ได้ชัดเจนขึ้น ลดภาระการอ่านของผู้ใช้

## Apply When

ใช้เป็นแนวทางในการเขียน System Prompt สำหรับ Agent ที่ต้องมีการทักทายหรือสรุปผล เพื่อให้เกิดความ Inclusive และรักษาความกระชับของเนื้อหา
