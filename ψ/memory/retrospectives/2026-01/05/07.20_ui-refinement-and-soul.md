## 1. Accomplishments (What we did)
- **Home Page Transformation**: นำ `MimiLoadingAvatar` (3D Soul) มาเป็นหัวใจหลักของหน้าแรก ทำให้แอปดูมีชีวิตชีวาและมีความเป็น "สายมู" มากขึ้น
- **Accessibility & Theme Polish**:
    - เปลี่ยนสี Hover จากชมพูเป็น **สีทอง (Accent)** เพื่อเพิ่ม Contrast และความหรูหรา
    - กำจัดสีเทามาตรฐาน (`muted-foreground`) ออกจากระบบ และแทนที่ด้วยโทนน้ำตาล/ทอง ตาม Design Token
- **Mobile Navigation**: เพิ่มปุ่ม "Back" ในหน้าประวัติสำหรับมือถือ และปรับปรุง Modal รายละเอียดไพ่ให้สวยงามขึ้น
- **User Identity**: ปรับปรุงการแสดงผล User ID ให้ย่อส่วนและสามารถคลิกเพื่อ Copy ได้ทั้งใน Navbar และ Profile page
- **3D Optimization**: ปรับปรุง `MimiLoadingAvatar` ให้แสดงผลเส้น Wireframe ด้านนอก (Shell) บนมือถือได้ชัดเจนเหมือน Desktop

## 2. Challenges & Learnings (What we found)
- **The Centering Struggle**: การจัดวางวัตถุ 3D (Canvas) ให้กึ่งกลางหน้าจอเป๊ะๆ ในทุกอุปกรณ์มีความซับซ้อนกว่าที่คิด โดยเฉพาะเมื่อต้องทำงานร่วมกับ Navbar และระบบ Animation ของ Tailwind
- **Fixed vs Absolute**: พบว่าการใช้ `fixed` ในโหมด Focus มักจะทำให้ Browser คำนวณจุดศูนย์กลางผิดพลาด (ชิดซ้ายบน) ซึ่งอาจเกิดจาก Conflict กับ CSS ตัวอื่นหรือการคำนวณ `transform` ที่ซ้อนทับกัน
- **Mobile Visibility**: เส้น Wireframe บางๆ อาจมองเห็นได้ยากบนมือถือเนื่องจากแสงสะท้อนและความละเอียดหน้าจอ การเพิ่ม Opacity และเปิด Antialiasing ช่วยแก้ปัญหานี้ได้
